Overview

Making Learning a Game

Bonzo is a digital education platform built around game-based learning principles — using reward loops, challenges, and interactive mechanics to make curriculum content more engaging for students.

The design system was built to balance educational rigour with game experience, ensuring that learning objectives were never lost within the game layer.
